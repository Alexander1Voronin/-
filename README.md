# Автоматизированная система складского учета
## Описание проекта

Программа для учета материальных ценностей на складе с использованием Windows Forms и C#. Система позволяет вести полный контроль над движением товаров, формировать отчеты и анализировать складские операции.

## Технологический стек

**Frontend:**

- **Windows Forms** 

- **C# .NET**

- **DataGridView** - для отображения данных

- **Chart** - для построения графиков

**Backend:**

- **SQL Server** - для хранения данных

- **LINQ** - для работы с базой данных

- **Entity Framework** (опционально)

## Основные функции

- **Управление товарами:**

- - Добавление/редактирование позиций

- - Учет прихода и расхода

- - Фильтрация по различным параметрам

- - Сортировка данных

- **Аналитические инструменты:**

- - Группировка данных по периодам

- - Вычисление итоговых сумм

- **Администрирование:**

- -Управление пользователями

- -Настройка прав доступа

- -Резервное копирование

## Структура базы данных

Система работает с двумя основными таблицами:

- **Товары** (номенклатура, единицы измерения, остатки)

- **Движения** (приход/расход, дата, количество, поставщик)

## Инструкция по установке

1. Установите Visual Studio с поддержкой Windows Forms

2. Настройте подключение к базе данных

3. Установите необходимые компоненты .NET Framework

4. Запустите исполняемый файл

## Требования к системе

- **ОС:** Windows 7 или новее

- **Память:** минимум 2 ГБ RAM

- **Место на диске:** 4 ГБ

- **.NET Framework:** версия 4.6 или выше
